# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In summary: be respectful, assume good faith, and focus on the work. Harassment, discrimination, and bad-faith behavior have no place here.

## Enforcement

Instances of unacceptable behavior may be reported by opening a GitHub issue or contacting the maintainers directly. All reports will be handled with discretion.
